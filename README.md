# NLO EW and QCD corrections to p p -> e+ ve mu+ mu- A at the LHC


## Prerequisites

* gfortran
* LHAPDF library
* RECOLA library
* HELAS library

## Installation

```
cd wza-nlo-ew
make

```

## Usage

Run the executable,
```
  ./epvemumua.x
  
```
type in a random seed (eg. '0') and press ```enter```
